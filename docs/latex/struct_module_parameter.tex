\doxysubsection{Detailed Description}
Module parameters. 

This is the main Lattice parameter struct. As as many of these to as you need to the {\ttfamily std\+::vector\texorpdfstring{$<$}{<}\mbox{\hyperlink{struct_module_parameter}{Module\+Parameter}}\texorpdfstring{$>$}{>}\& parameters} that is passed to the {\ttfamily Lattice\+Module\+Processor\+::create\+Parameters(std\+::vector\texorpdfstring{$<$}{<}\mbox{\hyperlink{struct_module_parameter}{Module\+Parameter}}\texorpdfstring{$>$}{>}\& parameters)} method. Each parameter added here will be accessible in the Lattice graph when you double click the processor module.
