\doxysubsection{Detailed Description}
Data from the host. 

One of these is passed to the {\ttfamily \mbox{\hyperlink{class_lattice_processor_module_ab6c881508a4469c69c499bf3c2aeb286}{Lattice\+Processor\+Module\+::process()}}} method each time is it called. This can be used to query everything from the current host BPM to whether or not the host is actually playing. (Note\+: only available in effect plugins)
