\doxysubsection{Detailed Description}
A simple host info struct. 

One of these is passed to the {\ttfamily \mbox{\hyperlink{class_lattice_processor_module_a1f76dc73529617fafe1a88a2bced0827}{Lattice\+Processor\+Module\+::process()}}} method each time is it called. This can be used to query everything from the current host BPM to whether or not the host is actually playing. (Note\+: only available in effect plugins)
